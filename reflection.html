Englewood Codes is a summer program where 25 teens in the community code to build websites using Html, CSS, Github, Bootstrap, and many others. In Englewood Codes we use code to build our own website. Englewood Codes is a positive place where we all work together in teams and learn to be leaders in the tech world. Coding is a very interesting thing but a very complex thing at the same time. Englewood Codes allows you to beb you and show your true colors through technology.
